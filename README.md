## Features
- Futuristic neon visuals (glow, particles)
- Power-ups: speed, invincibility, multiplier, shrink
- Moving obstacles and level progression
- Local high score
- Packaged build instructions (PyInstaller)

## Requirements
- Python 3.10+ (3.11  recommended)
- Pygame

## Quick start (development)

# create and activate virtualenv
python -m venv venv

# run
